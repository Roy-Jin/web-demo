<h1 align="center">RuoYiBoard</h1>

> 一个支持压感的在线白板，便于用户进行创作与表达。

## 🚀 快捷键表

| 功能 | 快捷键 |
| :-----: | :------: |
| 保存 | `Ctrl + S` |
| 打开 | `Ctrl + O` |
| 画笔 | `B` |
| 橡皮 | `E` |
